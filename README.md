Códigos pedidos para o HackerRank de Algoritmos e Estruturas de Dados 2 da UTFPR - PB em 2020

Nesse repositório se encontram os códigos que eu enviei ao HackerRank, esses são os originais dos últimos envios que fiz para cada código.

Para fazê-los, tive ajuda e sugestões principalmente dos colegas <a href="https://github.com/Proyol">Daniel Gustavo Favero</a>, <a href="https://github.com/joaoHenriqueMachado">João Henrique Pereira Machado</a>, <a href="https://github.com/cinglair"> Cinglair Augusto Capelo</a> e <a href="https://github.com/souzjfe">Jeferson Souza</a>, como também do professor da matéria <a href ="https://github.com/jefferson-oliva">Jefferson Tales Oliva</a>.

Qualquer pessa que queira utilizá-lo está livre para tal, de preferência, deixando os créditos do repositório; todavia, eu aconselho a não fazer a cópia deles, pois poderá acarretar em problemas para o próprio aluno, sugiro que tente entender a lógica por trás dos exercícios.

Obs: os códigos aqui estão completos em suas páginas, pois o HackerRank não permite o envio de vários arquivos para um exercício (ou eu não sabia como mandar dessa forma).

Qualquer dúvida, você pode me mandar um e-mail no seguinte endereço: ikvicari@gmail.com, que eventualmente eu responderei.
