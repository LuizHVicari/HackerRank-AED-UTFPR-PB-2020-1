# HackerRank-AED-2-2020/1

Códigos pedidos para o HackerRank de Algoritmos e Estruturas de Dados 2 da UTFPR-PB em 2020

Nesse repositório se encontram os códigos que eu enviei ao HackerRank, esses são os originais dos últimos envios que fiz para cada código.

Para fazê-los, tive ajuda e sugestões principalmente dos colegas Daniel Favero (Proyol), João Henrique Machado (joaoHenriqueMachado), Cinglair Capello e Jeferson Souza (Souzjef), como também do professor da matéria Jefferson Tales Oliva.

Alguns códigos neste repositório já se encontram comentados (para deixar mais limpo, comentei apenas as funções que eram necessárias para resolver os problemas das questões), os que ainda não estiverem, em breve serão atualizados.

Qualquer pessa que queira utilizá-loestá livre para tal, de preferência, deixando os créditos do repositório; todavia, é aconselho a não fazer a cópia deles, pois poderá acarretar em problemas para o próprio aluno, sugiro que tente entender a lógica por trás dos exercícios.

Obs: os códigos aqui estão completos em suas páginas, pois o HackerRank não permite o envio de vários arquivos para um exercício (ou eu não sabia como mandar dessa forma).

Qualquer dúvida, você pode me mandar um e-mail no seguinte endereço: ikvicari@gmail.com, que eventualmente eu responderei.
